차량 정보 등록

URL: http://localhost:8080/rental/api/register

정보 등록 요청:

curl -X POST -H "Content-Type: application/json" \
-d '{"category":"현대","manufacturer":"Toyota","model":"RAV4","year":"2025","status":"NEW"}' \
http://localhost:8080/rental/api/register

차량 정보 조회

URL: http://localhost:8080/rental/api/lookup

정보 조회 요청:

curl -G "http://localhost:8080/rental/api/lookup" --data-urlencode "searchType=현대"
curl -G "http://localhost:8080/rental/api/lookup" --data-urlencode "searchType=available"
curl -G "http://localhost:8080/rental/api/lookup" --data-urlencode "searchType=2022"
curl -G "http://localhost:8080/rental/api/lookup" --data-urlencode "searchType=쏘렌토"

차량 정보 수정

URL: http://localhost:8080/rental/api/update

정보 업데이트 요청:

curl -X PATCH 'http://localhost:8080/rental/api/update' -H 'Content-Type: application/json' \
-d '{"id": 2, "category": "미니"}'
